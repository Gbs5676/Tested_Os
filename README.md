
# Testes de Sistemas em WebAssembly

Aqui você encontrará ISOs de sistemas testados no modo Live, que funcionaram dentro do ambiente WebAssembly sem maiores problemas. É importante ressaltar que essas distribuições não estabelecem padrões fixos; portanto, fique ciente de que a execução pode depender de diversos fatores, como hardware, navegador e sistema operacional.

Os testes foram realizados em hardware mínimo, o que teoricamente garante a compatibilidade com a maioria dos dispositivos eletrônicos modernos disponíveis no mercado.
